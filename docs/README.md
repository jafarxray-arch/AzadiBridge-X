## 🚀 30 ثانیه تا دانلود

1. این ریپو را **Fork** کن
2. برو به **Actions** → **Azadi Smart Downloader**
3. لینک را بچسبان → **Run workflow**
4. فایل داخل **downloads/** ظاهر می‌شود
