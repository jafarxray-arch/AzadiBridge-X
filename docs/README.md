## 📖 English | انگلیسی

### What is this?

**CloudBridge** is a technical tool that uses GitHub Actions as cloud runners to download and archive media files from various public platforms. It's designed for developers, researchers, and archivists who need to preserve online content.

### Supported Platforms

`YouTube` | `Instagram` | `Twitter/X` | `Telegram (public channels)` | `Aparat` | `Spotify` | `SoundCloud` | `Reddit` | `TikTok` | `General Webpages`

### How It Works

1. Fork this repository to your GitHub account
2. Enable workflow write permissions
3. Run the desired workflow with your target URL
4. GitHub Actions downloads the file to your repository
5. You download it normally from GitHub

### Quick Start

**Step 1:** Click the **Fork** button at the top of this page

**Step 2:** Go to `Settings` → `Actions` → `General` → Select **Read and write permissions** → **Save**

**Step 3:** Go to `Actions` tab → Select a workflow → Click **Run workflow** → Enter your URL

**Step 4:** Find your file in the `downloads/` folder of your forked repository

### Available Workflows

| Workflow | Purpose |
|----------|---------|
| `Smart Downloader` | Automatically detects platform and downloads |
| `Web Archive` | Saves complete webpages as MHTML |
| `Split Large File` | Breaks files >95MB into smaller chunks |

## 📖 فارسی | Farsi

### این ابزار چیست؟

**کلودبریج** یک ابزار فنی است که از اکشن‌های گیت‌هاب به عنوان راننده ابری برای دانلود و آرشیو فایل‌های مدیا از پلتفرم‌های عمومی مختلف استفاده می‌کند. این ابزار برای توسعه‌دهندگان، پژوهشگران و آرشیوکنندگانی طراحی شده که نیاز به حفظ محتوای آنلاین دارند.

### پلتفرم‌های پشتیبانی شده

`یوتیوب` | `اینستاگرام` | `توییتر/ایکس` | `تلگرام (کانال‌های عمومی)` | `آپارات` | `اسپاتیفای` | `ساندکلود` | `ردیت` | `تیک‌تاک` | `صفحات وب عمومی`

### نحوه کار

1. این مخزن را در حساب گیت‌هاب خود فورک کنید
2. مجوز نوشتن اکشن‌ها را فعال کنید
3. ورکفلو مورد نظر را با لینک خود اجرا کنید
4. گیت‌هاب اکشن فایل را در مخزن شما دانلود می‌کند
5. شما فایل را به صورت معمولی از گیت‌هاب دانلود می‌کنید

### شروع سریع

**مرحله ۱:** دکمه **Fork** را در بالای این صفحه بزنید

**مرحله ۲:** بروید به `Settings` → `Actions` → `General` → گزینه **Read and write permissions** را انتخاب کنید → **Save**

**مرحله ۳:** بروید به تب `Actions` → ورکفلو مورد نظر را انتخاب کنید → **Run workflow** را بزنید → لینک خود را وارد کنید

**مرحله ۴:** فایل خود را در پوشه `downloads/` مخزن فورک شده خود پیدا کنید

### ورکفلوهای موجود

| ورکفلو | کاربرد |
|--------|--------|
| `Smart Downloader` | تشخیص خودکار پلتفرم و دانلود |
| `Web Archive` | ذخیره کامل صفحات وب به فرمت MHTML |
| `Split Large File` | تقسیم فایل‌های بزرگتر از ۹۵ مگابایت |
